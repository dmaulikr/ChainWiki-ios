Description
주요 기능:

1. 아르카나 필터 (레어, 직업, 무기, 소속, 주점)

2. 어빌리티별 아르카나 표시

3. 즐겨찾기 및 이름 검색 이용한 쉽고 빠른 아르카나 검색

4. 아르카나 정보 편집 기능

주의사항: 아르카나 정보는 일첸 위키 데이터를 불러 구글로 번역됐습니다. 따라서 수치가 존재하지 않을 수 있고, 단어 선택이 올바르지 않을 수 있습니다. 고칠 부분이 있으면 수정해 주세요!
